# E2E 테스트 시나리오 목록 

## 1. 주요 기능 테스트

### 1-1. 메뉴 관리 (Menu)

* 1-1-1. 신규 메뉴 카테고리 및 메뉴 항목 추가

  * 흐름

    1. 하단 탭에서 '메뉴' 선택
    2. 위젯 열기 → '분류 추가' 클릭
    3. 모달에서 분류명 입력 → '추가하기' 클릭
    4. 생성된 카테고리에서 '상품 추가' 클릭
    5. 메뉴 사진, 이름, 가격, 태그 입력 → '추가하기' 클릭

  * 예상 결과

    * 새 카테고리와 메뉴가 화면에 정상 표시

* 1-1-2. 메뉴 항목 수정 및 삭제

  * 흐름

    1. 수정할 메뉴 클릭
    2. 가격 수정 → '수정하기' 클릭
    3. 메뉴 클릭 → '삭제하기' 클릭

  * 예상 결과

    * 가격 변경 반영
    * 메뉴 항목 목록에서 삭제

* 1-1-3. 메뉴 카테고리 수정 및 삭제

  * 흐름

    1. '분류 수정' 클릭 → 카테고리 선택 → 새 이름 입력 → '수정하기' 클릭
    2. '분류 삭제' 클릭 → 삭제할 카테고리 선택 → '삭제하기' 클릭

  * 예상 결과

    * 카테고리 이름 변경
    * 카테고리와 하위 메뉴 삭제

---

### 1-2. 좌석 관리 (Table)

* 1-2-1. 신규 좌석 생성

  * 흐름

    1. '좌석' 탭 선택
    2. '좌석 추가' 클릭 → 캔버스 중앙 확인 → '추가하기' 클릭
    
  * 예상 결과

    * 좌석 생성, 새로고침 후에도 위치 유지

* 1-2-2. 좌석 위치 및 크기 변경

  * 흐름

    1. '좌석 수정' 클릭 → 편집 모드
    2. 좌석 선택 → 위치 이동 및 크기 조절
    3. '수정하기' 클릭

  * 예상 결과

    * 위치와 크기 변경 상태 저장

* 1-2-3. 좌석 QR코드 확인 및 삭제

  * 흐름

    1. 좌석 클릭 → 정보 모달 확인
    2. QR 코드 확인
    3. '좌석 삭제' 클릭 → 삭제

  * 예상 결과

    * 정보 모달 및 QR 코드 정상 표시
    * 좌석 캔버스에서 삭제

---

### 1-3. 주문 처리 (Order)

* 1-3-1. 신규 주문 접수 및 완료 처리

  * 흐름

    1. 테스트 DB에 신규 주문 생성
    2. '주문' 탭 선택 → '접수' 탭 확인
    3. 주문 카드 정보 확인 → '조리 완료' 클릭
    4. '완료' 탭 클릭

  * 예상 결과

    * '접수' 탭 주문 카드 사라짐
    * '완료' 탭에서 비활성화 상태로 표시

---

### 1-4. 실시간 요청 처리 (Request)

* 1-4-1. 고객 요청 알림 확인 및 처리

  * 흐름

    1. 테스트 DB에 고객 요청 생성
    2. 요청 알림 확인 → 테이블 번호/요청 내용 확인
    3. 알림 메시지 'x' 버튼 클릭

  * 예상 결과

    * 알림 즉시 표시
    * 'x' 클릭 시 알림 사라짐

---

## 2. 추가 테스트 (예외, UI/UX, 실시간)

### 2-1. 예외 및 경계 값 테스트

* 2-1-1. 데이터 미존재

  * 흐름: 메뉴, 좌석, 주문 데이터 없는 초기 상태에서 각 탭 진입

  * 예상 결과: 안내 문구("표시할 데이터가 없습니다") 표시

* 2-1-2. 입력 값 유효성 검사

  * 흐름: 필수 입력 없이 메뉴/카테고리 추가, 가격 음수 입력

  * 예상 결과: 알림(alert)으로 유효성 메시지 표시

* 2-1-3. 동시성 테스트

  * 흐름: 두 브라우저/탭에서 동일 메뉴 가격 동시 수정

  * 예상 결과: DB 최종 반영 가격 일관성 확인

---

### 2-2. 반응형 및 UI/UX 테스트

* 2-2-1. 모바일 및 다양한 화면 크기

  * 흐름: 브라우저 창 크기 축소 → 모바일 환경 시뮬레이션

  * 예상 결과: 모달/위젯 표시 여부, 레이아웃 깨짐 없음, 가로/세로 전환 정상

* 2-2-2. 위젯 및 모달 동작

  * 흐름: 위젯 열기/닫기 반복, 모달 열려 있을 때 탭 전환, 배경 클릭

  * 예상 결과: 애니메이션 부드럽게 동작, 탭 전환 방지, 배경 클릭 시 모달 닫힘

---

### 2-3. 실시간 기능 심화 테스트

* 2-3-1. 다중 요청 및 주문 처리

  * 흐름: 여러 테이블에서 동시에 주문/요청 발생

  * 예상 결과: 모든 주문/요청 누락 없이 표시, 처리 시 알림 정상 사라짐

* 2-3-2. 네트워크 연결 상태

  * 흐름: 네트워크 오프라인 → 온라인 복구

  * 예상 결과: 연결 상태 아이콘 색상 변경, 오프라인 동안 변경된 데이터 동기화
